# Earn-Points
# Cube Push Puzzle (Unity Prototype)

## About
Cube Push Puzzle is a simple Unity physics prototype.

The player controls a cube using WASD keys and pushes small cubes into a target area.  
When all cubes are placed inside the target zone, the level is completed.

## Gameplay
- WASD movement
- Physics-based cube pushing
- Simple win condition (all cubes in target area)

## Built With
- Unity Engine
- C#
- Visual Studio

## Purpose
This project was created to learn basic Unity mechanics such as:
- Physics interactions
- Player movement
- Simple game logic

## Status
Prototype (single level, early version)
